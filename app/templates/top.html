はじめに

  1. 研究内容
    2. 人が画像から受ける印象の分析や、印象に応じた画像を生成する研究です。
  2. 研究責任者
    3. 慶應義塾大学大学院理工学研究科 橋本敦史
  3. 実験目的
    4. 人が画像から受ける印象のデータセット作成のための実験です。
  4. 実験時間
    5. 説明約5分+実験約50分+休憩5分の約1時間です。
  5. 実験環境
    6. なるべく通信環境の整っている場所で、自分のパソコンで行ってください。
  6. 実験期間
    7. 別途連絡した期日までに行ってください。
  7. 実験対象者
    8. 本実験は、人が画像から受ける印象を色で表すものとなっております。大変申し訳ありませんが、色覚異常のある方は本実験に参加することができません。ご了承ください。
  8. 実験内容
    9. パソコンの画面に画像と2つの色が順に一瞬現れます。深く考えず、パッと見た瞬間の印象のみを手がかりに、ご⾃⾝が感じた印象に近い⾊の⽅向に、スライドバーを調整していただきます。
  9. 協力に対する謝礼について
    10. 謝礼金は1000円です。
  10. 個人情報の取り扱い
    11. 実験によって得られたデータについては、統計的に処理した結果のみを学会等で発表し、個別的なデータを個人が特定可能な形で公開することはありません。また、個人情報を含むすべてのデータは外部に漏洩することのないように匿名化して厳重に管理します。
  11. 行動規範
    12. スクリーンショットやスクリーンキャプチャにより画像をダウンロードすることやデータの収集、および、悪用などをしてはいけません。
本研究の問い合わせ先
研究責任者：慶應義塾大学大学院理工学研究科橋本敦史
e-mail：tezuka2021-theme5-group@keio.jp

実験手順

  1. あらかじめメールにてお送りした user name と password でログインしてください。
  2. Nextボタンを押してください。
  3. Nextボタンを押すと、最初に画像が一瞬現れます。
  4. 次に、二つの色が一瞬現れます。
  5. そして、スライドバーが表示されます。深く考えずパッと見た瞬間の印象のみを手がかりに、ご⾃⾝が感じた印象に近い⾊の⽅向に、スライドバーを調整してください。
    必ずスライドバーを動かし値(Value)が０にならないようにして下さい。 数秒経過後、自動的にページが切り替わります。
  6. 以上の作業を繰り返し行ってください。（画像を⾒逃した、もしくは画像が正しく写らなかった場合、スライドバーを押さずに次の画⾯を待ってください。）

それでは、あらかじめメールにてお送りした user name と password でログインし、実験を開始してください。
